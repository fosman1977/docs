<Note>
View and manage barrister profiles and information.
</Note>

## Accessing Profiles

Navigate to **Barristers** in the sidebar to see all barrister profiles.

## Profile Information

Each profile shows:

### Professional Details
- Name and title
- Call date
- Areas of practice
- Jurisdictions

### Current Status
- Availability
- Utilisation percentage
- Pending leads

### Performance
- Lead acceptance rate
- Engagement score
- Billing metrics

### Specialisations
- Tagged practice areas
- Expertise levels

## Profile Views

### List View
Quick overview of all barristers with key stats.

### Card View
Visual cards with profile photos and summary.

### Detailed View
Full profile with all information.

## Actions

From barrister profiles you can:
- Assign leads
- View calendar
- Check availability
- See performance history
