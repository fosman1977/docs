View and manage case records and instruction history.
</Note>

## What Are Case Records?

Case records track instructions after leads convert:
- Matter details
- Barrister assignment
- Key dates
- Outcome

## Viewing Records

Access from:
- Lead details (after conversion)
- Client profile
- Barrister profile

## Information Tracked

- Matter reference
- Client details
- Assigned barrister
- Value and billing
- Status and outcome

## Using Records

Case records help with:
- Understanding client history
- Tracking barrister experience
- Generating reports
- Practice development analysis
