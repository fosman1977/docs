Import historical data from Lex using CSV files.
</Note>

## Overview

The CSV import feature allows you to bring historical contact data from Lex into the Chambers Platform.

## Preparing Your Data

Export from Lex:
1. Open Lex
2. Navigate to Contacts
3. Export as CSV
4. Save the file

## Import Process

<Steps>
  <Step title="Navigate to Import">
    Admin → Data Import → CSV Import
  </Step>
  <Step title="Upload file">
    Select your CSV file
  </Step>
  <Step title="Map fields">
    Match Lex fields to platform fields
  </Step>
  <Step title="Review">
    Check the preview
  </Step>
  <Step title="Import">
    Confirm and run import
  </Step>
</Steps>

## Field Mapping

Common mappings:

| Lex Field | Platform Field |
|-----------|----------------|
| Company Name | Client Name |
| Contact Name | Contact |
| Email | Email |
| Phone | Phone |

## After Import

Review imported records:
- Check for duplicates
- Verify data accuracy
- Link to existing records if needed

<Warning>
Always backup your data before large imports.
</Warning>
