## What Are Follow-ups?

Follow-ups are scheduled reminders to:
- Chase lead responses
- Check in with clients
- Follow up on opportunities
- Complete pending actions

## Creating Follow-ups

Follow-ups can be created:
- Automatically (system-generated)
- Manually from any lead or client

## Follow-up Types

| Type | Description |
|------|-------------|
| Lead Follow-up | Chase barrister response |
| Client Follow-up | Check in with client |
| Opportunity Follow-up | Post-event actions |
| General | Any other reminder |

## Managing Follow-ups

Access from **Follow-ups** in the sidebar.

### Views
- Today's follow-ups
- Overdue
- Upcoming
- Completed

### Actions
- Complete
- Reschedule
- Delegate
- Cancel
