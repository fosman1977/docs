<Note>
How to work with and manage follow-up tasks efficiently.
</Note>

## Daily Workflow

Start each day by:
1. Checking today's follow-ups
2. Addressing any overdue items
3. Reviewing upcoming tasks

## Task Management

### Prioritising
- High priority - Action today
- Medium priority - Action this week
- Low priority - Action when possible

### Bulk Actions
Select multiple follow-ups to:
- Mark as complete
- Reschedule together
- Reassign

## Automation

The system automatically creates follow-ups for:
- Leads pending response over 24 hours
- Overdue barrister responses
- Upcoming opportunity deadlines

## Best Practices

- Address overdue items first
- Set realistic due dates
- Add context in notes
- Complete items promptly

<Tip>
Review follow-ups at the start and end of each day.
</Tip>
