<Note>
The messaging system allows clerks and barristers to communicate internally about leads, clients, and opportunities.
</Note>

## Overview

The messaging system provides:

- Internal communication between clerks and barristers
- Message threading by lead, client, or opportunity
- Draft management
- Message templates for common responses
- AI-assisted draft generation

## Accessing Messages

Navigate to **Messages** in the sidebar to access your inbox.

### Message Views

| View | Description |
|------|-------------|
| Inbox | All received messages |
| Sent | Messages you've sent |
| Drafts | Unsent draft messages |
| All | Complete message history |

## Sending Messages

<Steps>
  <Step title="Click New Message">
    Click the "New Message" button
  </Step>
  <Step title="Select Recipients">
    Choose one or more recipients from the user list
  </Step>
  <Step title="Link to Record (Optional)">
    Optionally link the message to a lead, client, or opportunity
  </Step>
  <Step title="Compose Message">
    Write your message content
  </Step>
  <Step title="Send or Save Draft">
    Click "Send" to deliver or "Save Draft" to continue later
  </Step>
</Steps>

## Message Templates

Save time with pre-defined templates for common messages:

### Using Templates

1. Click "Use Template" when composing
2. Select from available templates
3. Customise the content as needed
4. Send the message

### Creating Templates

Templates are managed by administrators at **Admin** → **Settings** → **Message Templates**

## AI-Assisted Drafts

<Info>
Use AI to help draft professional messages based on context.
</Info>

### How to Use

1. When composing a message linked to a lead or client
2. Click "AI Draft" button
3. AI generates a suggested message based on:
   - Lead/client context
   - Recent activity
   - Communication history
4. Review and edit as needed
5. Send when satisfied

## Message Threading

Messages linked to the same record are automatically grouped:

- View conversation history in context
- All participants can see the thread
- Related messages appear on the lead/client detail page

## Notifications

You'll receive notifications for:

- New messages received
- Replies to your messages
- Messages on leads you're managing

Configure notification preferences at **Settings** → **Notifications**

<Tip>
Enable email notifications to stay updated when away from the platform.
</Tip>
