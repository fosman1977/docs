## Overview

The Team Overview provides a comprehensive view of all barristers you manage, their current status, and workload.

## How to Access

Navigate to: **Dashboard** → **Team** → **Overview**

## Key Features

### Barrister List

View all barristers with key information:
- Current availability status
- Utilisation percentage
- Pending lead count
- Recent activity

### Availability Status

| Status | Meaning |
|--------|---------|
| Available | Ready for new work |
| Busy | High utilisation |
| Away | On leave or holiday |
| Unavailable | Not taking new work |

### Quick Actions

- View full profile
- Check calendar
- See assigned leads
- Contact barrister

## Tips & Best Practices

- Check team overview before assigning leads
- Note barristers in quiet periods for proactive assignment
- Keep availability statuses updated
