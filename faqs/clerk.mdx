<Note>
Common questions from clerks using the platform.
</Note>

<AccordionGroup>
  <Accordion title="How do I create a new lead?">
    Click Leads in the sidebar, then click New Lead. Fill in the required fields and click Create.
  </Accordion>
  <Accordion title="How does AI suggestion work?">
    After creating a lead, click Get Suggestions. The AI analyses the lead against barrister profiles and suggests matches.
  </Accordion>
  <Accordion title="What if multiple barristers accept the same lead?">
    You'll receive a notification and can choose which barrister gets the instruction.
  </Accordion>
  <Accordion title="How do I chase a response?">
    Set up follow-up reminders, or contact the barrister directly. The system can send automated reminders.
  </Accordion>
  <Accordion title="Can I delegate my work when on leave?">
    Yes, set up a delegation in Settings → Delegations before you go.
  </Accordion>
  <Accordion title="How do I import contacts from Lex?">
    Export from Lex as CSV, then use Admin → Data Import → CSV Import.
  </Accordion>
  <Accordion title="Why aren't AI suggestions appearing?">
    Check that AI features are enabled and you haven't exceeded daily limits. Also ensure the lead has practice area set.
  </Accordion>
  <Accordion title="How do I run reports?">
    Go to Reports, select the report type, set your parameters, and click Generate.
  </Accordion>
</AccordionGroup>
