## Common Issues

<AccordionGroup>
  <Accordion title="Financial data isn't showing">
    - Check Xero integration status in Admin
    - Verify the barrister has a Xero contact mapping
    - Last sync may have failed - try manual refresh
  </Accordion>
  <Accordion title="Client data seems outdated">
    - Client data is stored directly in the platform
    - If you've imported data via CSV, check the import completed successfully
    - Use the CRM search to verify the client record exists
  </Accordion>
  <Accordion title="New client isn't appearing">
    - Check the client was created successfully
    - Verify you have the correct permissions to view the client
    - Try refreshing the page or clearing your browser cache
  </Accordion>
  <Accordion title="Integration shows 'Disconnected'">
    - Re-authenticate in Admin → Integrations
    - Permissions may have been revoked
    - Check with your administrator
  </Accordion>
</AccordionGroup>

## Manual Sync

Force a Xero sync by going to Admin → Integrations and clicking "Sync Now" on the Xero integration.
