## Xero Issues

<AccordionGroup>
  <Accordion title="Xero authentication expired">
    Xero tokens expire after 30 days. Re-authenticate in Admin → Integrations → Xero.
  </Accordion>
  <Accordion title="Invoices not appearing">
    - Check the barrister has a Xero contact mapping
    - Verify invoices are approved in Xero
    - Review mapping in Admin → Xero Mappings
  </Accordion>
  <Accordion title="Financial data not syncing">
    - Check Xero connection status in Admin → Integrations
    - Verify your Xero subscription is active
    - API rate limits may have been reached - wait and retry
  </Accordion>
  <Accordion title="Barrister revenue not updating">
    - Ensure the barrister is mapped to a Xero contact
    - Check if invoices are correctly tagged in Xero
    - Review the sync log for any errors
  </Accordion>
</AccordionGroup>

## General Troubleshooting

<AccordionGroup>
  <Accordion title="Integration shows disconnected">
    - Navigate to Admin → Integrations
    - Click on the affected integration
    - Re-authenticate with your credentials
  </Accordion>
  <Accordion title="Sync seems stuck">
    - Check the last sync time
    - Try clicking "Sync Now" to force a refresh
    - If issues persist, contact support
  </Accordion>
</AccordionGroup>

## Getting Help

If issues persist, contact support@33cllaw.com with:
- Which integration is affected
- Error messages shown
- When the issue started
