## Supported Browsers

The Chambers Platform works best with modern, up-to-date browsers:

| Browser | Minimum Version |
|---------|----------------|
| Google Chrome (recommended) | Version 90+ |
| Microsoft Edge | Version 90+ |
| Mozilla Firefox | Version 88+ |
| Safari | Version 14+ |
| Opera | Version 76+ |

## Not Supported

<Warning>
The following browsers are not supported:
- Internet Explorer (any version)
- Browsers older than versions listed above
- Text-only browsers
</Warning>

## Required Browser Settings

- JavaScript must be enabled
- Cookies must be enabled for authentication
- Pop-up blocker may need exceptions for some features
- LocalStorage must be available

## Mobile Support

The platform is responsive and works on mobile devices:

| Platform | Browser | Version |
|----------|---------|---------|
| iOS | Safari | Version 14+ |
| Android | Chrome | Latest version |
| Samsung | Samsung Internet | Version 14+ |

## Recommended Screen Resolution

For the best experience:

| Device | Resolution |
|--------|------------|
| Desktop | 1280x720 or higher |
| Laptop | 1024x768 minimum |
| Mobile | Standard smartphone screens supported |

<Tip>
Keep your browser updated for the best security and performance.
</Tip>
