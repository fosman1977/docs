Multiple ways to get help when using the Chambers Platform.

## Contextual Help System

The platform includes a built-in help system that provides relevant documentation based on your current page.

### Using the Help Button

1. Look for the Help button (?) in the header or use **Shift + ?**
2. Click to open help for your current page
3. The help panel shows relevant documentation
4. Click links to navigate to more detailed guides

## Documentation

Comprehensive documentation is available covering:

- Getting Started guides for new users
- Role-specific guides (Barrister, Clerk, Management)
- Feature documentation for each platform area
- FAQs and troubleshooting guides
- Training and onboarding checklists

## Reporting Issues

When reporting a problem, please include:

1. What page you were on
2. What action you were trying to perform
3. Any error messages shown
4. Screenshots if possible
5. Your browser and device type

<Tip>
Most common issues have solutions in the FAQs section - check there first!
</Tip>
