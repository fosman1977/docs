## What is 2FA?

Two-factor authentication adds an extra layer of security by requiring:
1. Your password
2. A code from your phone

## Enabling 2FA

<Steps>
  <Step title="Go to Security Settings">
    Settings → Security → Two-Factor Authentication
  </Step>
  <Step title="Click Enable">
    Start the setup process
  </Step>
  <Step title="Scan QR code">
    Use an authenticator app (Google Authenticator, Authy, etc.)
  </Step>
  <Step title="Enter verification code">
    Confirm with a code from the app
  </Step>
  <Step title="Save backup codes">
    Store these securely for recovery
  </Step>
</Steps>

## Supported Apps

- Google Authenticator
- Authy
- Microsoft Authenticator
- 1Password

## Recovery

If you lose your device:
- Use backup codes
- Contact administrator
- Reset via verified email

<Warning>
Store backup codes securely - they're your only recovery option.
</Warning>
