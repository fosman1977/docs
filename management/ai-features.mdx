## AI Capabilities

The platform uses AI (powered by OpenAI GPT-4o-mini) for:

### Lead Matching
Suggests optimal barristers for each lead based on:
- Specialisations
- Experience
- Availability
- Past performance

### Business Insights
Analyses data to identify:
- Trends and patterns
- Risks and opportunities
- Performance anomalies

### Reporting
AI-assisted report generation and analysis.

## Privacy & Security

### Data Handling
- Client names are tokenised
- No PII sent to AI
- Data is anonymised

### Usage Limits
Daily limits apply to AI features to manage costs.

## Configuration

AI features can be enabled/disabled from:
**Admin** → **Settings** → **AI Configuration**

Options include:
- Enable/disable features
- Set usage limits
- Configure notification thresholds
