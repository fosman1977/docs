## What's Logged

The audit log records:
- User logins
- Data changes
- Permission modifications
- System access
- Configuration changes

### Log Entry Details

Each entry shows:
- Timestamp
- User who performed action
- Action type
- Affected entity
- Changes made

## Exporting

Export logs for:
- Compliance reporting
- Security reviews
- Investigation

Formats: CSV, PDF

## Retention

Audit logs are retained for:
- 12 months by default
- Longer for compliance needs
- Configurable in settings
