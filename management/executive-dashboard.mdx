## Dashboard Overview

The Executive Dashboard provides at-a-glance visibility of:
- Key performance indicators
- Trend analysis
- Alerts and insights
- Action items

## Key Widgets

### KPI Summary
Top-line metrics including:
- Total leads this period
- Conversion rate
- Revenue YTD
- Utilisation average

### Trend Charts
Visual representation of:
- Lead volume over time
- Conversion trends
- Revenue trends
- Utilisation patterns

### AI Insights
Automatically generated insights about:
- Unusual patterns
- Risks identified
- Opportunities spotted

### Action Items
Things requiring management attention:
- Pending approvals
- System alerts
- Performance flags

## Customisation

Adjust the dashboard:
- Select which widgets to display
- Set date ranges
- Filter by practice area
- Compare periods
