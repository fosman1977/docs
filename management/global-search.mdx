<Note>
Find anything in the platform with global search.
</Note>

## Using Search

Access search:
- Click search icon in header
- Press **Ctrl/Cmd + K**

## What's Searchable

- Leads (title, description, client)
- Clients (name, company)
- Barristers (name)
- Opportunities (title)
- Documents

## Search Tips

### Basic Search
Type keywords and press Enter.

### Filters
Add filters to narrow results:
- Type (lead, client, etc.)
- Date range
- Status

### Keyboard Shortcuts
- **↑↓** Navigate results
- **Enter** Open selected
- **Esc** Close search

## Results

Results show:
- Type icon
- Title/name
- Brief context
- Quick actions
