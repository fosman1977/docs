## Available Roles

### Barrister
- View personal dashboard
- Respond to leads
- Participate in opportunities
- View own metrics

### Clerk
- Full lead management
- Opportunity management
- CRM access
- Report generation

### Management
- All clerk permissions
- Executive dashboards
- User administration
- System configuration

## Assigning Roles

Users can have multiple roles:
1. Go to User Management
2. Select user
3. Add/remove roles
4. Save changes

## Role Switching

Users with multiple roles:
- Choose role at login
- Switch via user menu
- Actions logged by role

## Custom Permissions

Some permissions can be customised:
- Report access
- Financial visibility
- Admin functions

<Info>
Role changes take effect immediately - user may need to refresh.
</Info>
