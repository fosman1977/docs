## Settings Categories

### General
- Chambers name and details
- Time zone
- Date format
- Default currency

### Leads
- Default assignment options
- Auto-follow-up timing
- Status workflow

### Notifications
- System-wide notification settings
- Email templates
- Digest schedules

### Display
- Default views
- Branding options
- Custom labels

## Making Changes

1. Select the category
2. Modify settings
3. Click Save
4. Changes apply immediately

<Info>
Some settings require system restart to take effect.
</Info>
