## User List

View all users with:
- Name and email
- Role(s)
- Status
- Last login

## Adding Users

<Steps>
  <Step title="Click Add User">
    Click the "Add User" button
  </Step>
  <Step title="Enter details">
    Name, email, and initial role
  </Step>
  <Step title="Send invitation">
    System sends login credentials
  </Step>
</Steps>

## Editing Users

Click on any user to:
- Update details
- Change roles
- Reset password
- Deactivate account

## User Statuses

| Status | Meaning |
|--------|---------|
| Active | Can log in |
| Inactive | Cannot log in |
| Pending | Invitation sent |

## Bulk Actions

Select multiple users to:
- Change roles
- Activate/deactivate
- Export list
