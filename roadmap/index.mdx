## Roadmap Status Legend

- <span style="color: #10b981;">● In Planning</span> - Feature scoped, specifications in progress
- <span style="color: #3b82f6;">● In Development</span> - Active coding and testing
- <span style="color: #8b5cf6;">● In Pilot</span> - Testing with select users
- <span style="color: #22c55e;">● Launched</span> - Available to all users
- <span style="color: #f59e0b;">● Blocked</span> - Awaiting decisions or dependencies
