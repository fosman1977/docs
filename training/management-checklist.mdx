## Before First Login

<Steps>
  <Step title="Receive credentials">
    Get admin access from system administrator
  </Step>
  <Step title="Review integrations">
    Understand Xero financial integration setup
  </Step>
</Steps>

## First Login

<Steps>
  <Step title="Log in">
    Use the credentials provided
  </Step>
  <Step title="Change password">
    Set a strong password
  </Step>
  <Step title="Set up 2FA">
    Enable two-factor authentication
  </Step>
</Steps>

## System Review

<Steps>
  <Step title="Review users">
    Check all user accounts
  </Step>
  <Step title="Verify integrations">
    Confirm Xero is connected and syncing
  </Step>
  <Step title="Check settings">
    Review system configuration
  </Step>
</Steps>
